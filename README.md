# منصة تراؤف للرعاية (نسخة B2B الاحترافية)

منصة متكاملة مصممة خصيصاً لجمعية "تراؤف" لتفعيل المسؤولية الاجتماعية (CSR) مع الشركاء التجاريين بكفاءة وشفافية.

## المتطلبات الأساسية
- Node.js (v18+) للواجهة الأمامية
- Python 3.9+ للواجهة الخلفية
- Docker & Docker Compose (مستحسن لتشغيل قاعدة البيانات)

## 1. تشغيل قاعدة البيانات (PostgreSQL)
لتسهيل التشغيل والتجربة، قمنا بتوفير بيئة Docker متكاملة تقوم بإنشاء قاعدة البيانات وتعبئتها آلياً ببيانات تجريبية (Mock Data) تحاكي الواقع السوقي السعودي.

1. افتح الطرفية (Terminal) في مسار المشروع الرئيسي.
2. قم بتشغيل الأمر:
   ```bash
   docker-compose up -d
   ```
3. ستعمل قاعدة البيانات على المنفذ `5432` وستكون جاهزة للاستخدام.

## 2. تشغيل الواجهة الخلفية (Backend - FastAPI)
يحتوي الخادم على واجهات برمجة التطبيقات (APIs) لربط وتغذية النظام بالبيانات.

1. انتقل لمسار `backend`:
   ```bash
   cd backend
   ```
2. (اختياري) قم بإنشاء بيئة وهمية: `python -m venv venv` ثم فعلها `. venv/Scripts/activate`
3. ثبّت المكتبات المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```
4. شغل الخادم:
   ```bash
   uvicorn main:app --reload
   ```
5. يمكنك فحص التوثيق التفاعلي للـ API عبر: `http://localhost:8000/docs`

## 3. تشغيل الواجهة الأمامية (Frontend - Next.js)
1. افتح نافذة طرفية أخرى وانتقل لمسار `frontend`:
   ```bash
   cd frontend
   ```
2. (في حال كانت أول مرة) ثبت الاعتماديات:
   ```bash
   npm install --legacy-peer-deps
   ```
3. شغل بيئة التطوير محلياً:
   ```bash
   npm run dev
   ```
4. تفضل بزيارة: `http://localhost:3000`

---
> **ملاحظة أمنية:** البيانات المدرجة في `seed.sql` هي بيانات وهمية ومصممة لغرض اختبار العرض فقط ولا تحتوي على معلومات حقيقية لأفراد أو جهات فعلية تحقيقاً لسياسة الخصوصية والأمان.
